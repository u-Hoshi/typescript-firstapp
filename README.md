# ユーザー管理アプリ

TypeScritp と React を使用した初めてのアプリです。

## 使用技術

- React Hooks
- TypeScript
- React-Router
- Chakra UI
- axios(json の読み込み)
- jsonPlaceholder json 形式でデータを取得するために API サーバ
- dog.ceo json 形式で画像を取得するために API サーバ

元々画像は source.unsplash.com から取得するつもりだったが、axios を使って画像のリロード処理がしたかったので json 形式で取得する方向に変更
